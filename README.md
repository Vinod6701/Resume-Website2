
# K. Vinod Kumar - AWS DevOps Portfolio

A modern, professional portfolio built with React, TypeScript, and Tailwind CSS.

## Features
- **Responsive Design**: Works on mobile, tablet, and desktop.
- **AWS Focus**: Highlights expertise in Cloud and DevOps tools.
- **PDF Generation**: "Download CV" button optimized for browser printing.
- **CI/CD**: Automated deployment to GitHub Pages via GitHub Actions.

## Development

To run this project locally:

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Build for production: `npm run build`

## Deployment

This project is configured to deploy automatically to GitHub Pages. Simply push your code to the `main` branch of your GitHub repository.
